# Welcome to Fren Pet

Fren Pet is an indie game on the base blockchain developed by [@caduveloso](https://twitter.com/cadu_veloso) and [@surfcoderepeat](https://twitter.com/surfcoderepeat), we will keep adding a feature a week as long as players enjoy and are active.

## Download the app

You can download the mobile app on [frenpet.xyz](https://frenpet.xyz), it will create a crypto wallet for you to use in the game. You will need to send in some base ETH to get started.

## Minting a Pet

You can mint a Fren Pet on the app, the minting price is decided by a dutch auction that runs for 10 minutes, it starts at 2000 $FP and goes down slowly until it reaches 100 $FP on the 10th minute.

Anyone can mint a pet at any point of the dutch auction, once a pet is minted a new dutch auction automatically starts.

This process helps find the fair market value for a pet at all times.

All tokens used to mint pets are burned.
