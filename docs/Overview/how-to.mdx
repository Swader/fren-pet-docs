# Welcome to Fren Pet

Fren Pet is an indie game on the base blockchain developed by [@caduveloso](https://twitter.com/cadu_veloso) and [@surfcoderepeat](https://twitter.com/surfcoderepeat) that aims to explore true ownership on the blockchain, we will keep adding a feature a week as long as players enjoy and are active.

## Download the app

You can download the mobile app on [frenpet.xyz](https://frenpet.xyz), it will create a crypto wallet for you to use in the game. You will need to send in some base ETH to get started.

## Minting a Pet 

Minting a pet costs 100 $FP, but the next time someone else mints a pet, 100 $FP are being sent back to you.
