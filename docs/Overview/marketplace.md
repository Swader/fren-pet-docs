# 市场

游戏内的市场提供各种物品，从化妆品到非常有用的物品，比如盾牌可以在被攻击时保护你，或者保险可以在TOD耗尽且宠物死亡时自动复活你的宠物。

商店中的所有物品都以$FP进行交易，根据一个线性倾斜曲线进行定价，每个物品的价格变动可以在商店中看到。

购买物品时有10%的费用，这些费用会转给推荐人（见下文）。

注意：推荐系统已被移除，但如果你是早期玩家并有推荐，你仍然会获得这些费用。
